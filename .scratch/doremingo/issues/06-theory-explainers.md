# 06 — Theory explainers between levels

Status: ready-for-agent

## Parent

`.scratch/doremingo/PRD.md`

## What to build

Friendly plain-language theory cards shown before a level's first question on first entry: what's new in this level and why it sounds the way it does (e.g. what a key/home note is, what the cadence does, what makes Mi sound "sunny"). Written for zero musical education; formal terms (tonic, scale degree) introduced gently alongside the game's words. Content lives as plain data (theory content module), one explainer per level, always skippable and re-readable from the level map.

## Acceptance criteria

- [ ] Every level in the current progression has an explainer; shows automatically only on first entry
- [ ] Skippable mid-read; re-openable from the level map for any unlocked level
- [ ] Copy uses glossary vocabulary, no unexplained jargon, no note names as answers
- [ ] Adding a new level's explainer requires only a data change
