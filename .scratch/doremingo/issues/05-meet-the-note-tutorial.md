# 05 — Meet-the-Note and first-run tutorial

Status: ready-for-agent

## Parent

## What to build

The interactive first-run tutorial (~60s, no text walls): hear the Cadence ("this is home"), tap Do/Mi/Sol to hear each, hear one Resolution, then flow into Level 1's first real Question. Each level that unlocks a new Degree opens with a Meet-the-Note moment for it (hear it, tap it, hear its Resolution). Tutorial replayable from the home screen; everything skippable.

## Acceptance criteria

- [ ] Fresh profile (empty storage) lands in the tutorial, which ends in a real Level 1 Question
- [ ] New-degree levels show Meet-the-Note for the new Degree on first entry only
- [ ] Tutorial replayable from home screen; all steps skippable by tap
- [ ] Copy assumes zero musical education and uses glossary vocabulary
