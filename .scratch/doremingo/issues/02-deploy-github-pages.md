## What to build

Get the game live on a public URL. Create the GitHub repo, enable Pages, ensure asset paths work under the Pages subpath, add a README with the live link and local dev instructions (static server, tests).

## Acceptance criteria

- [ ] Game playable at the public GitHub Pages URL on desktop and a real or emulated mobile browser
- [ ] Piano samples load over the network with acceptable size (~1–2 MB total)
- [ ] README documents live URL, local serving, and `node --test`
- [ ] Pushing to the default branch updates the site
